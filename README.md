# schema-studierendendaten
Dies ist das XML-Schema für die Archivierung von Studierendendaten im Rahmen des Projekts Digital Campus. Das Schema basiert auf dem XML-Schema von nestor zur «Archivierung von Studierendendaten aus Fachverfahren - Version 1.0», passt es aber an Schweizer Gegebenheiten an und erfüllt alle erforderlichen Best Practices von eCH-0018.
